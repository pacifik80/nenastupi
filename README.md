# nenastupi

Сервис проверки надёжности работодателя (MVP).

## Быстрый старт (локально)

1) Скопируйте пример окружения:

```bash
cp .env.example .env
```

2) Укажите `TELEGRAM_BOT_TOKEN` и поменяйте пароли.

3) Запустите:

```bash
docker compose up --build
```

Сервис:
- API: http://localhost:8000
- Admin UI: http://localhost:8000/admin (Basic Auth)
- Neo4j Browser: http://localhost:7474

## Основные эндпоинты

- `POST /api/check` — проверка компании по названию/ИНН/ОГРН
- `GET /health` — проверка инфраструктуры
- `GET /admin` — административная панель

## MVP источники

- ФНС (ЕГРЮЛ) — статус и основные сведения
- ЕФРСБ — признаки банкротства (минимальный парсер)
- Google News RSS — новости (по запросу)

## Примечания

- Для новостей используется RSS Google News без обхода ограничений.
- В отсутствие данных из источников используется демо-набор тестовых компаний из ТЗ.
